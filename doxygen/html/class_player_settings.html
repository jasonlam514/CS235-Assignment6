Detailed Description

Allows you to change settings of player 1 and player 2.

\file PlayerSettings.Java

Author
Callum Hazelton 709161-A4, Gavin Bailey 711036-A5, Chak Yan Lam 667271-A6
Date
18/04/2014

Change the player names, player colours, whether players are human/ai and difficulty of ai. Changes to the class involved separating the action listeners to comply with code conventions. Other changes include adding the substance to the resume game button to allow it load the file.
